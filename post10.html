1. Firstly download R in https://cran.cnr.berkeley.edu/ and install R in a location you know.There are many mirrors to choose in fact.
2. In extensions (Ctrl+Shift+S) of VS code download Powershell and R
3. In setting (Ctrl+",") of VS code, go to extensions -> r -> Rterm:Windows and set path to the location of R.exe where you installed (e.g. D:\R\R-3.5.1\bin\R.exe)
4. There are lots of stuff to configure default path of Powershell. open your r script and Powershell terminal in VScode. Type

get-executionpolicy

in Powershell and it will return restricted. To change the  permission to remotesigned, open Powershell with administrator privilege and type

set-executionpolicy remotesigned

5. Create dictionary and file: C:\Users\XXX\Documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1
6. In powershell, 
echo "Set-Location D:\" > $Profile
7. Test your first script by typing  .\Rscript path-of-your-script in Powershell, e.g.

.\Rscript e:\Untitled-1.r
